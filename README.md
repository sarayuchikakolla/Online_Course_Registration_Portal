# Online Course Registration Portal

## Project Overview

The Online Course Registration Portal is a responsive web application developed using HTML, CSS, Bootstrap 5, and JavaScript. The portal allows students to explore available courses, view course details, register online, learn about the institute, and contact the training center.

The website is fully responsive and works across desktop, tablet, and mobile devices.

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Ensure all images are placed inside the images folder.
4. Open index.html in any web browser.
5. Navigate through the website using the navbar.
